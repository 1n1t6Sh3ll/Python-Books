### HTML & CSS

### JavaScript

1. **[Eloquent JavaScript by Marijn Haverbeke](https://eloquentjavascript.net/)**
   - Free online book
   - Interactive code examples
   - Comprehensive coverage of JavaScript fundamentals

2. **[You Don't Know JS (Book Series) by Kyle Simpson](https://github.com/getify/You-Dont-Know-JS)**
   - Deep dive into JavaScript

   - Covers ES6+ features

   - Fun, beginner-friendly introduction
   - Interactive examples
   - Great for absolute beginners
